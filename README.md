# Roseline Rose Couture

A six-page preview site for **Roseline Rose Couture** — a small couture maison for women and men, refined, unhurried, made by hand and guided by faith.

**Live at:** https://aaemmanuel-created.github.io/roseline-rose-couture/

> *"She is clothed with strength and dignity; she can laugh at the days to come."* — Proverbs 31:25

## Pages

| Page | Purpose |
| --- | --- |
| `index.html` | Maison — home, manifesto, featured collection, values, lookbook teaser |
| `collections.html` | Congo in Naija — the full 17 pieces, filterable |
| `lookbook.html` | Editorial spreads — six frames from Volume I |
| `about.html` | The Atelier — founder, philosophy, three vows, timeline |
| `journal.html` | Letters from the maison |
| `contact.html` | Contact form & appointment info |

## Structure

```
.
├── index.html
├── collections.html
├── lookbook.html
├── about.html
├── journal.html
├── contact.html
└── styles.css        ← shared house styles
```

Pure HTML and CSS — no build step, no framework. Open `index.html` directly in any browser, or host on any static host (GitHub Pages, Netlify, Vercel, Cloudflare Pages).

## House style

- **Palette** — ivory, ink, rose, blush
- **Type** — *Italiana* for display, *Cormorant Garamond* italic for accents, *Marcellus* for labels, *EB Garamond* for body
- **Fonts** are loaded from Google Fonts and require an internet connection

## Images

Image placeholders use CSS gradients (rose, blush, dark, ivory). To swap in photography, replace each `<div class="imgph">` with an `<img>` tag pointing at your file.
